用Play! 2框架搭建自己的博客(0)

Apr 8th, 2013

3月份的时候用Play! 2框架搭建了一个博客，包括前台和后台，但是由于没有合适的云平台可以放，又没mon买服务器，所以只能压箱底了…先po几张图

技术架构：Play! 2框架 + MySQL数据库 + Ebean ORM.

虽然界面有点丑(好吧，本人缺少美感),虽然功能很简单(只是实现了对文章的CRUD和分类的CRUD)，虽然还有很多要修改的东西……

But，这是我是学了play!框架后完全自己做出来的东西，终于把理论转换为实践了。很开心很感动的其实～ （PS:因为play框架在国内并不流行，能找到的学习资料真少，不然就是旧版本的，学习的时候都是看play！官网上的文档，遇到问题也都只能stakoverflow，oschina上问或自己慢慢的尝试）
